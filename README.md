jQuery Date Picker
================================

jQuery Date Picker is developed base on jQuery plugin ([uxsolutions-bootstrap-datepicker-plugin](https://github.com/uxsolutions/bootstrap-datepicker/blob/master/docs/index.rst)), and adopted the Style Portal.

-------------------------
